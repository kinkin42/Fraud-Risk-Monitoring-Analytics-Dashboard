# Credit Card Fraud Risk Analysis – Power BI Dashboard

## Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze **credit card fraud risk patterns** across transactions.
The dashboard enables stakeholders to identify **high-risk fraud types**, **geographical hotspots**, **transaction categories**, and **temporal trends**, supporting data-driven fraud prevention and risk mitigation.

---

## Objectives

* Analyze fraudulent transaction patterns by **fraud type**
* Evaluate **risk-level distribution** (Low, Medium, High, Critical)
* Identify **high-risk transaction categories**
* Detect **state-wise fraud concentration**
* Monitor **monthly fraud trends**
* Provide an intuitive, filter-driven analytics experience

---

## Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **Microsoft Bing Maps (Power BI Map Visual)**
* **CSV / Structured Transaction Dataset**

---

## Dashboard Features

### Key Performance Indicators (KPIs)

* **Fraud Rate (%)**
* **Total Fraudulent Transactions**
* **Critical Risk Transactions**

### Interactive Filters

* Fraud Type
* State
* Merchant Name

### Visual Analytics

* **Fraud Type Analysis** – Transaction count by fraud category
* **Risk-Level Distribution** – Donut chart showing Low, Medium, High, Critical risk amounts
* **Category-wise Transaction Amount** – Comparative bar chart
* **Geographical Analysis** – State-wise fraud distribution map
* **Monthly Trend Analysis** – Line/area chart showing transaction count over months

## Key Insights

* **Card Not Present (CNP)** fraud is the most frequent fraud type
* **Critical-risk transactions** account for a significant monetary value
* **E-commerce and Electronics** categories show higher fraud exposure
* Certain **states exhibit concentrated fraud activity**, indicating regional risk patterns
* **Fraud peaks toward year-end**, suggesting seasonal trends
